## 机器作曲·遗传算法
### 数据集
数据输入：从具有相同节拍的若干个去、乐曲中选取十个长度相等（例如4小节）的片段。   

> 目前：使用基于MuseScore的music21将midi文件转化为不同声道的音符和和弦，取第一个乐器（没有乐器则取旋律）,设定范围（C-4到B#6）取每个和弦中在范围内的一个音符，构成音符序列。
- 值得注意的是：   
  一些路径需要自己设定；暂时还没有封装成函数；数据也可以作为音符流存储。

  https://web.mit.edu/music21/doc/
